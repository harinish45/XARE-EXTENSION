Usage Instructions

To use the glowing border effect in your project:

  1. Include glowing-border.css in your HTML
  2. Include glowing-border.js before your closing body tag
  3. Add data-glow attribute to any element
  4. Customize with optional data attributes:
    • data-glow-spread - Gradient spread angle (default: 40)
    • data-glow-proximity - Activation distance (default: 64)
    • data-glow-border-width - Border thickness (default: 3)
    • data-glow-blur - Blur amount (default: 0)
    • data-glow-inactive-zone - Center dead zone (default: 0.01)
    • data-glow-variant - Color variant: "default" or "white"
